# Minh Truong - Portfolio

A modern, responsive portfolio website built with Next.js, TypeScript, and Tailwind CSS.

## Tech Stack

- **Framework:** Next.js 16 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS v4
- **Fonts:** Syne (display), DM Sans (body)
- **Images:** next/image

## Features



- Dark/Light mode with localStorage persistence
- Responsive design (mobile-first)
- Smooth scroll navigation
- Scroll-triggered animations
- SEO optimized (meta tags, Open Graph, sitemap, robots.txt)
- Accessibility (ARIA labels, skip-to-content, reduced-motion support)

## Sections

- **Hero** - Profile introduction with CTAs
- **About** - Biography and education
- **Experience** - Skills grid (Frontend + Backend)
- **Projects** - Featured project cards
- **Contact** - Email, GitHub, Facebook links

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start

# Lint
npm run lint
```

## Deployment

Deploy to Vercel with zero configuration:

```bash
npm i -g vercel
vercel --prod
```

Or connect your GitHub repository to Vercel for automatic deployments.
